## Run Locally

Clone the project

```bash
  git clone git@github.com:zablon-oigo/cafe.git
```

Go to the project directory

```bash
  cd cafe
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
